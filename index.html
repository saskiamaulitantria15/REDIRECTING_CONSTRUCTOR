Main Constructor dengan Radius

Main constructor merupakan constructor utama pada kelas Circle

yang menerima nilai radius secara langsung saat objek dibuat. Constructor

ini menjadi pusat inisialisasi data lingkaran dan digunakan oleh

constructor lain yang melakukan redirect.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

class Circle {
  final double radius;

  Circle(this.radius);

  @override
  String toString() {
    return 'Lingkaran dengan radius $radius';
  }
}

void main() {
  var lingkaran = Circle(10);

  print(lingkaran);
}

Redirecting dari Diameter

Redirecting constructor dari diameter adalah constructor yang digunakan

untuk membuat objek Circle berdasarkan nilai diameter. Nilai

diameter akan diubah menjadi radius terlebih dahulu, kemudian diteruskan ke

constructor utama menggunakan this().

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

class Circle {
  final double radius;

  Circle(this.radius);

  Circle.dariDiameter(double diameter)
      : this(diameter / 2);

  @override
  String toString() {
    return 'Lingkaran dengan radius $radius';
  }
}

void main() {
  var lingkaran = Circle.dariDiameter(20);

  print(lingkaran);
}

Redirecting dari Keliling

Redirecting constructor dari keliling adalah constructor yang digunakan

untuk membuat objek Circle berdasarkan nilai keliling lingkaran.

Radius dihitung dari keliling menggunakan rumus matematika, lalu

hasilnya diteruskan ke constructor utama.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

import 'dart:math';

class Circle {
  final double radius;

  Circle(this.radius);

  Circle.dariKeliling(double keliling)
      : this(keliling / (2 * pi));

  @override
  String toString() {
    return 'Lingkaran dengan radius $radius';
  }
}

void main() {
  var lingkaran = Circle.dariKeliling(62.8);

  print(lingkaran);
}

Redirecting dari Luas

Redirecting constructor dari luas adalah constructor yang digunakan untuk

membuat objek Circle berdasarkan nilai luas lingkaran. Radius diperoleh dari

hasil perhitungan luas lingkaran dan kemudian digunakan untuk memanggil

constructor utama melalui this().

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

import 'dart:math';

class Circle {
  final double radius;

  Circle(this.radius);

Implementasi Lengkap Redirecting Constructor pada Class Circle

Class Circle adalah class yang digunakan untuk merepresentasikan objek lingkaran.

Class ini memiliki satu constructor utama yang menerima nilai radius dan tiga redirecting constructor

yang dapat membuat objek berdasarkan diameter, keliling, dan luas. Redirecting constructor menggunakan sintaks :

this(...) untuk mengarahkan proses pembuatan objek ke constructor utama sehingga kode menjadi lebih ringkas dan

menghindari duplikasi.

//Nama: Saskia Mauli Tantria 
//NIM: 251410034
//Kelas: SI2A

import 'dart:math';

class Circle {
  final double radius;

  // Main constructor
  Circle(this.radius);

  // Redirecting constructor dari diameter
  Circle.dariDiameter(double diameter)
      : this(diameter / 2);

  // Redirecting constructor dari keliling
  Circle.dariKeliling(double keliling)
      : this(keliling / (2 * pi));

  // Redirecting constructor dari luas
  Circle.dariLuas(double luas)
      : this(sqrt(luas / pi));

  @override
  String toString() {
    return 'Lingkaran dengan radius $radius';
  }
}

void main() {
  var lingkaran1 = Circle(10);
  var lingkaran2 = Circle.dariDiameter(20);
  var lingkaran3 = Circle.dariKeliling(62.8);
  var lingkaran4 = Circle.dariLuas(314);

  print('Main Constructor');
  print(lingkaran1);

  print('\nRedirecting dari Diameter');
  print(lingkaran2);

  print('\nRedirecting dari Keliling');
  print(lingkaran3);

  print('\nRedirecting dari Luas');
  print(lingkaran4);
}

Kesimpulan

- Redirecting Constructor adalah constructor yang digunakan untuk memanggil constructor lain yang berada

dalam class yang sama dengan menggunakan sintaks : this(...).

- Constructor ini membantu menghindari penulisan kode yang berulang pada proses inisialisasi objek.

- Redirecting Constructor tidak memiliki body karena hanya bertugas meneruskan parameter ke constructor yang dituju.

- Constructor ini sering digunakan untuk membuat beberapa cara pembuatan objek dalam satu class.

- Penggunaannya membuat kode lebih ringkas, terorganisir, dan lebih mudah dipelihara.

- Redirecting Constructor mendukung prinsip DRY (Don't Repeat Yourself) dengan mengurangi duplikasi kode.
